⚡ QUICK SETUP (15 minutes total):

STEP 1: Get Cloudflare Turnstile (5 min)
   → https://dash.cloudflare.com/
   → Add Site → Copy Site Key + Secret Key
   → Add to backend/.env: TURNSTILE_SECRET_KEY=...
   → Add to frontend/.env: VITE_TURNSTILE_SITE_KEY=...

STEP 2: Get EmailJS (5 min)
   → https://www.emailjs.com/
   → Add Service → Add Template → Get Credentials
   → Add to backend/.env:
      EMAILJS_SERVICE_ID=...
      EMAILJS_TEMPLATE_ID=...
      EMAILJS_PUBLIC_KEY=...

STEP 3: Deploy Backend (3 min)
   → https://render.com/
   → New Web Service → Connect GitHub
   → Root Dir: Drishti Kavach/backend
   → Build: npm install, Start: npm run dev
   → Add all env vars from backend/.env
   → Copy deployment URL

STEP 4: Deploy Frontend (2 min)
   → https://vercel.com/
   → New Project → Import GitHub
   → Root Dir: Drishti Kavach/frontend
   → Build: npm run build
   → Add all env vars from frontend/.env
   → Update VITE_API_URL to backend URL

═══════════════════════════════════════════════════════════════════════════════
